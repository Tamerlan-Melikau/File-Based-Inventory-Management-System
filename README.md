# Product Inventory Management System

A C++ console application for managing product inventory using binary files. The project demonstrates practical work with file handling, data structures, sorting algorithms, and search algorithms.

## Key Features

* Create and manage inventory databases
* Add, edit, delete, and view product records
* Store data in binary files
* Sort products by:

  * Quantity (Quick Sort)
  * Weight (Selection Sort)
  * Price (Insertion Sort)
* Search products by name and price
* Generate inventory reports
* Export query results to text files
* Generate random test data

## Technologies

* C++
* Binary File I/O (`FILE*`)
* Custom sorting algorithms
* Linear and Binary Search
* Console-based UI

## What This Project Demonstrates

* Working with binary files and persistent storage
* Implementation of classic algorithms without STL containers
* Data validation and user input handling
* Structured program design using functions and custom data types
* Report generation and file management

## Build

```bash
g++ main.cpp -o inventory.exe
```
